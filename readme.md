# 俄罗斯电商AI自动化运营系统

## 项目定位
面向俄罗斯跨境电商卖家的AI运营操作系统。
支持 Wildberries / Ozon / Yandex Market 三平台统一管理，
AI驱动广告投放优化、SEO内容生成、库存采购、财务分析，
最终面向多品牌多店铺SaaS商业化输出。

## 技术栈
- 后端：Python 3.11 + FastAPI
- 数据库：MySQL 8.0（ECS本地部署）
- 缓存/队列：Redis + Celery
- AI：DeepSeek（分析）/ Kimi（长文档）/ GLM（俄语文案）
- 通知：企业微信群机器人 + 应用消息
- 前端：React + Ant Design
- 部署：阿里云ECS Ubuntu 22.04
- 代码管理：GitHub
